2- Ejecutando turtle desde terminal o jupyter¶

Terminal de Vs-Code¶

Primero debemos activar una terminal de python, si estamos en VS-Code con el comando python, ptyhon nos mostrara las 3 flechas ">>>" para indicar que nos esta escuchando el interprete de python

Todo lo que escribamos despues de las >>> lo guardara python en memoria mientras el terminal este activo.

Entonces le ensearemos todas las funciones de turtle

damos enter y ahora simplemnte le damos culquiera de los comandos de la libreria de turtle y vemos como aparece la ventana

Recuerde que este modo no guarda los procedimientos que se van ejecutando. Usted debe guardarlos en alugn archivo aparte para generar despues sus movimientos personalizados
