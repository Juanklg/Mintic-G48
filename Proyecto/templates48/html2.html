Este proyecto consiste en una aplicación dirigida principalmente a jóvenes, pero cualquier persona podrá hacer uso de ella, en la cual los usuarios podrán interactuar de forma anónima con otros usuarios con la intención de recibir un apoyo emocional, una opinión, un consejo etc.
Esta app también contará con personas especializadas en esta área como psicólogos, pero deberá de haber una forma de pago si se quiere hacer uso de este servicio. En la aplicación los usuarios contarán con una mascota y también se encontrará espacios de relajación y de entretenimiento para garantizar la mejor comodidad y calidad al usuario.
